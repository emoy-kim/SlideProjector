# SlideProjector

 ![slideprojector](https://user-images.githubusercontent.com/17864157/63649776-03cb3800-c77d-11e9-8a1f-4e6db8ccfa1f.gif)
  
  
  
## Test Environment
  * Windows 10
  * Visual Studio 2017
  * Surface Book 2
  
## Library Dependencies
  * OpenCV
  * OpenGL

## Keyboard Commands
  * **Up Arrow**: move forward
  * **Down Arrow**: move backward
  * **Left Arrow**: move left
  * **Right Arrow**: move right
  * **w key**: move up
  * **s key**: move down
  * **i key**: main camera and projector reset
  * **l key**: light turn on/off
  * **r key**: replay projector when video was projected
  * **enter key**: project an image/video
  * **q/ESC key**: exit

## Mouse Commands
  * **Main camera**: moving with mouse left button clicked
  * **Projector**: moving with mouse left button clicked pressing *the left control key*
